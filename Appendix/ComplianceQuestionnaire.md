# Appendix: GDPR Compliance Questionnaire

## Q1. Does your application collect any personal data (e.g., name, IP, or location)?
- **Options:**
    - Yes → go to Q2
    - No → **Result:** It appears GDPR may not apply to this project. Learn why IP addresses still count as personal data.

## Q2. Is the stored data limited to only the strictly necessary minimum?
- **Options:**
    - Yes → go to Q3
    - No → **Result:** Data Minimization Gap: You are storing more data than required.

## Q3. Is there an automated cleanup process to delete data once it's no longer needed?
- **Options:**
    - Yes → go to Q4
    - No → **Result:** Storage Limitation Gap: Manual deletion processes are a compliance risk.

## Q4. Do you ensure data is not reused for marketing without new consent?
- **Options:**
    - Yes → go to Q5
    - No → **Result:** Purpose Limitation Gap: Data is being reused without justification.

## Q5. Is sensitive data protected via encryption or strict access controls?
- **Options:**
    - Yes → go to Q6
    - No → **Result:** Security Gap: Data must be protected from unauthorized access.

## Q6. Can users download a copy of their data in a readable format?
- **Options:**
    - Yes → go to Q7
    - No → **Result:** Access Right Gap: Missing a way for users to export their data.

## Q7. Does your system support a complete 'Right to Erasure' (to be forgotten)?
- **Options:**
    - Yes → go to Q8
    - No → **Result:** Erasure Right Gap: Systems must support a technical 'Right to be Forgotten'.

## Q8. Does your application handle 'Special Category' data (e.g., health data)?
- **Options:**
    - Yes → go to Q9
    - No → no immediate result

## Q9. Is the consent mechanism clear, separate, and easy to withdraw?
- **Options:**
    - Yes → go to Q10
    - No → **Result:** Consent Gap: Bundled consent is invalid for sensitive data.

## Q10. Do you provide a technical way for users to rectify (fix) inaccurate data?
- **Options:**
    - Yes → end
    - No → **Result:** Rectification Gap: Users cannot technically correct their information.
